Operators

+ - * / %

It's just math. 
// Return an em based value for suggested pixels

@function ems ( $size, $context: 16px ) {
    @return ( $size / $context ) * 1em;
}

.lead {
    font-size:ems(20px);
}
Output 

.lead {
    font-size: 1.25em;
}

Parent Selector

Parent selector 
.some-div {
    background-color:$red;
}
.some-div .some-child {
    background-color:$blue;
}

.some-child a{
    color:#eeeeee;
}
Parent selector 
.some-div {
    background-color:$red;

    & .some-child {
        background-color:$blue;

        & a {
            color:#eeeeee;    
        }
    }
}

Nesting

Group related declarations together

Write less code.

Plain CSS 
nav ul{ 
    margin:0;
    padding:0;
    list-style: none;
}

nav li {
    display: inline-block;
}

nav a {
    display: block;
    padding: 5px 10px; 
    text-decoration: none;
}
Nesting 
nav { 
    ul {
        margin:0;
        padding:0;
        list-style: none;   
    }
    
    li {
        display: inline-block;

        a {
            display: block;
            padding: 5px 10px; 
            text-decoration: none;
        }
    }
}

    Inheritence

    Don't repeat yourself.

    Horizontal lists 
    
%nav {
    list-style: none;
    padding: 0;
    margin: 0;
}
    %nav-item,
    %nav-link {
        display: block;
    }

%nav-horizontal {
    @extend %nav;
    @extend %cf;
}

                        
    .tab-list {
    @extend %nav-horizontal;
}
nav.nav--primary {
    @extend %nav-horizontal;
}
nav.nav--footer {
    @extend %nav-horizontal;
}
    Output 
    .tab-list, nav.nav--primary, nav.nav--footer {
    list-style: none;
    padding: 0;
    margin: 0;
}
